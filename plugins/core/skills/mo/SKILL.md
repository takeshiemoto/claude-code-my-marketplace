# mo

引数で渡された Markdown ファイルのパスを `mo` コマンドでブラウザに開く。`--open` フラグを必ず付けて、既存サーバー稼働中でも必ずブラウザを前面に開く。

## 使用方法

```sh
/core:mo <markdownファイルのパス>
```

引数で渡されたパスをそのまま `mo --open` に渡す。複数ファイル指定にも対応する。

## 実行フロー

1. 引数が空なら「開きたいMarkdownファイルのパスを指定してください」と促し、終了する
2. `Bash` で `mo --open <引数>` を実行する
3. コマンドの出力（サーバーURL等）をそのまま結果として提示する

## 注意

- `--open` は必須。これがないと、既存 `mo` サーバーへのファイル追加時にブラウザが開かず、スキルの目的（ブラウザでプレビュー）を満たさない
- パスにスペースが含まれる場合はダブルクォートで囲んで実行する
- 余計なオプション（`--port` `--target` 等）は付けない。ユーザーが明示した場合のみ追加する
