# mo

引数で渡された Markdown ファイルのパスを `mo` コマンドに渡してブラウザで開く。`mo` は既定でバックグラウンドサーバーを起動しブラウザを自動で開くため、追加処理は不要。

## 使用方法

```sh
/takeshiemoto:mo <markdownファイルのパス>
```

引数で渡されたパスをそのまま `mo` に渡す。複数ファイル指定にも対応する。

## 実行フロー

1. 引数が空なら「開きたいMarkdownファイルのパスを指定してください」と促し、終了する
2. `Bash` で `mo <引数>` を実行する
3. コマンドの出力（サーバーURL等）をそのまま結果として提示する

## 注意

- パスにスペースが含まれる場合はダブルクォートで囲んで実行する
- 既に `mo` サーバーが起動していれば、既存セッションにファイルが追加される（mo の標準動作）
- 余計なオプション（`--port` `--target` 等）は付けない。ユーザーが明示した場合のみ追加する
